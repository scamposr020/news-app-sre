### Contributing to News App (SRE Academy Project)
Thanks for showing interest in contributing to the News App! Every improvement here is not only about code — it’s also about practicing reliability, observability, and collaboration skills.
All kinds of contributions are welcomed!: new features, dashboards, alerts, documentation, or even small changes that make the project easier to use and understand.

1. Getting Started

    - Fork the Repository
    - Head over to https://github.com/scamposr020/SRE and click Fork.

2. Clone Your Fork
    
    ` git clone https://github.com/your-username/SRE.git `
    ` cd SRE`

3. Create a Branch
       
    `git checkout -b improvement/your-change`

4. Make Your Changes
    - Keep the code clean and consistent with the project style.
    - Add or adjust tests if needed.
    - Run the app locally to confirm everything works as expected.

5. Commit and Push
   
      `git add . `
      
      `git commit -m "Brief description of your change"`
      
      `git push origin improvement/your-change`

6. Open a Pull Request
   
    - Go back to the original repo.
    - Click Compare & pull request.

Explain clearly what you changed and why — remember, this is also a learning project, so context helps others follow along.
